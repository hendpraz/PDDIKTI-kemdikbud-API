# pddiktigo

Go scraper pipeline for PDDIKTI Kemdikbud data.

## Requirements

- Go 1.22 or newer.

## Usage

Run from the repository root:

```bash
go run ./pddiktigo
```

Useful options:

```bash
go run ./pddiktigo --limit 10
go run ./pddiktigo --delay 50
go run ./pddiktigo --output pddiktigo/mahasiswa_itb.txt
```
